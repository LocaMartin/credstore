## Positioning Copy

CredStore is built for teams that cannot place password vault contents in a cloud database. Vaults
stay local, sync happens directly between trusted devices, and commercial licensing works offline
through signed license tokens.

The server signs licenses. It does not hold your passwords, biometric data, or vault keys.
