How Offline Validation Works

CredStore signs the license payload server-side and embeds only the public verification key in the app. Users can activate premium features offline after receiving the QR or text key.

Anti-Sharing Model

The token is tied to the CredStore account identity. A copied QR fails in a different vault identity.
